WONDER HALL 4.0 — GITHUB CONNECTION GUIDE

Wonder Hall can publish Parent Wing changes directly to GitHub.

ONE-TIME GITHUB SETUP

1. Sign in to GitHub on your desktop.
2. Open GitHub Settings.
3. Open Developer settings.
4. Open Personal access tokens.
5. Choose Fine-grained tokens.
6. Create a new token.
7. Give it access only to the wonder-hall repository.
8. Under Repository permissions, set Contents to Read and write.
9. Create and copy the token.

CONNECT WONDER HALL

1. Open Wonder Hall on the desktop.
2. Open Parent Wing / Connected Curator.
3. Open Publish Wonder Hall.
4. Enter:
   Username: kittykat71
   Repository: wonder-hall
   Branch: main
   Website data file: resources.json
5. Paste the fine-grained token.
6. Click Test and Connect.

PUBLISH

1. Make changes in Parent Wing.
2. Open Publish Wonder Hall.
3. Click Publish Directly to GitHub.
4. Wait for the success message.
5. Refresh Wonder Hall on the phone and tablet after GitHub Pages finishes updating.

SECURITY

- Use a fine-grained token restricted to only the wonder-hall repository.
- Grant only Contents: Read and write.
- The token is stored in sessionStorage, so closing the browser removes it.
- Do not share or publish the token.
- You can revoke the token at any time from GitHub Settings.
